## Local Development

```sh
bin/dev
```

Then open <http://127.0.0.1:4000/blog/>.

If port `4000` is already in use, `bin/dev` will choose the next available port
and print the preview URL.

In VS Code, use `Terminal > Run Task... > Jekyll: Serve with LiveReload`.
